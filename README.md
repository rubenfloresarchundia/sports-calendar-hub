# Sports Calendar Hub

Dynamic sports calendar generator for iPhone, Apple Calendar, Google Calendar, Outlook and CarPlay.

## Goal

Create automatically updated iCalendar (.ics) files for sports events.

## First phase

Tennis player calendars, starting with:

- Jannik Sinner
- Carlos Alcaraz

The architecture will support adding more ATP and WTA players later from configuration.

## Future sports

- Football / Soccer
- NFL
- MLB
- Formula 1
- Tennis tournaments
